# Cassette::Rack

VCR front operation

## Installation

Add this line to your application's Gemfile:

```
gem 'cassette-rack'
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install cassette-rack
```

## Usage

Add this line to your `config.ru`:

```
run CassetteRack::Engine
```
